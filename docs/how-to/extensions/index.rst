Extensions
==========

This section contains guides for extensions. For a list of extensions, consult the
:ref:`extensions reference<reference-extensions>`.

Adding extensions to your snap requires a few simple changes. Extensions that aren't
ready for production require a change to your host's environment.

- :ref:`how-to-use-an-extension`
- :ref:`how-to-enable-experimental-extensions`

On the command line, you can gather details of what extensions are available, as well as
print the exact changes that an extension makes to your project before building.

- :ref:`how-to-list-extensions`
- :ref:`how-to-expand-extensions`

Some extensions have special requirements and need extra configuration.

- :ref:`how-to-use-the-env-injector-extension`
- :ref:`how-to-use-the-gnome-extension`
