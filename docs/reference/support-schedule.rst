Support schedule
================

Major versions of Snapcraft come with the same 12-year long term support (LTS)
commitment shared by many Canonical products.

In a supported major version of Snapcraft, the latest minor version receives direct
support.

Each major version of Snapcraft supports the latest core and two previous cores
available at time of release, as well as one future core. For example, Snapcraft 8 was
released to support core24, so it also supports core20, core22, and core26.
